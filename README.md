# ChartBuilder
ChartBuilder is a simple tool that converts Excel files into clear evaluation charts. Just upload your data, choose your chart type, and get visual results instantly—no complex setup required.
To use the program, you just need to run it and provide the path to your Excel file. The program will read your data, generate evaluation charts based on the content, and save them as image files on your desktop.

Once the process is complete, you'll find multiple chart files created directly on your desktop—ready to use, share, or include in your reports. No complex setup or technical skills required.
